Design Airline Reservation System Low Level Design :-

#Functional Requirements:
1. User Registration and Authentication:
   - Users should be able to create an account using their email and password.
   - Implement login and logout functionality.
   - Password recovery option via email.
2. Flight Search and Booking:
    - Users should be able to search for flights based on origin, destination, date, and number of passengers.
    - Display available flights with details such as flight number, departure and arrival times, duration, and price.
    - Allow users to select a flight and proceed to booking.
    - Collect passenger details (name, age, gender, passport number).
    - Provide seat selection options.
    - Implement payment gateway integration for processing payments.
3. Booking Management:
    - Users should be able to view their booking history.
    - Allow users to cancel or modify bookings.
    - Send email notifications for booking confirmations, cancellations, and reminders.
4. Admin Panel:
    - Admins should be able to add, update, or delete flight details.
    - View all bookings and manage user accounts.
    - Generate reports on bookings, cancellations, and revenue.
5. Reviews and Ratings:
    - Allow users to leave reviews and ratings for flights.
    - Display average ratings and reviews for each flight.

// Factory Pattern for Email + PushInbox + SMS Notification
interface Notification {
    void sendNotification(String message, String recipient);
}
class EmailNotification implements Notification {
    public void sendNotification(String message, String recipient) {
        // Implement email sending logic
    }
}
class SMSNotification implements Notification {
    public void sendNotification(String message, String recipient) {
        // Implement SMS sending logic
    }
}
class PushInboxNotification implements Notification {
    public void sendNotification(String message, String recipient) {
        // Implement push inbox sending logic
    }
}
class NotificationFactory {
    public static Notification createNotification(String type) {
        if (type.equalsIgnoreCase("EMAIL")) {
            return new EmailNotification();
        } else if (type.equalsIgnoreCase("SMS")) {
            return new SMSNotification();
        } else if (type.equalsIgnoreCase("PUSHINBOX")) {
            return new PushInboxNotification();
        }
        }
}
